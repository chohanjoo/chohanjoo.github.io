:one: 버블 정렬

정렬의 우선순위가 가장 낮은 data를 맨 뒤로 보낸다.

~~~c
for(int i=0;i<5-1;++i)
	for(int j=0;j<5-1-i;++j){
		if(array[j] > array[j+1]){
      int temp = array[j];
      array[j] = array[j+1];
      array[j+1] = temp;
		}
  }
~~~

i=1 -> 57318 57318 53718 53178
i=2 -> 53178 35178 31578
i=3 -> 31578 13578
i=4 -> 13578
~~~

~~~c
for(int i=1;i<5;++i){
~~~
